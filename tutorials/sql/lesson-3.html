🔍 Lesson 3: SELECT Queries (Retrieving Data)

⏱️ Estimated time: 20 minutes | Difficulty: Beginner

The Foundation

The SELECT statement is used to retrieve data from a database. It is the command you will use most often in SQL.

Table of Contents

  • Basic SELECT
  • ORDER BY (Sorting)
  • LIMIT & OFFSET
  • DISTINCT (Remove Duplicates)

Basic SELECT

-- Get all columns
SELECT * FROM users;

-- Get specific columns
SELECT name, email FROM users;

-- Alias columns
SELECT name AS "Full Name", age AS "Years" FROM users;

ORDER BY (Sorting)

SELECT * FROM users ORDER BY name ASC;    -- A to Z
SELECT * FROM users ORDER BY age DESC;     -- Highest first
SELECT * FROM users ORDER BY age DESC, name ASC; -- Multiple

LIMIT & OFFSET

SELECT * FROM users LIMIT 10;           -- First 10 rows
SELECT * FROM users LIMIT 10 OFFSET 20; -- Rows 21-30 (pagination)

DISTINCT (Remove Duplicates)

SELECT DISTINCT city FROM users;  -- Unique cities only
