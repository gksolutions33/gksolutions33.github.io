Your First C Program

Create a file called hello.c and type this:

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

Let's Break It Down Line by Line:

  • #include <stdio.h> ‚Äî Includes the Standard Input/Output library (gives us printf)
  • int main() ‚Äî The main function where your program starts. Every C program must have one!
  • printf("Hello, World!\n") ‚Äî Prints text to the screen. \n means new line
  • return 0; ‚Äî Tells the operating system "I finished successfully"
  • { } ‚Äî Curly braces define a block of code

Compiling & Running

# Step 1: Compile
gcc hello.c -o hello

# Step 2: Run
./hello          # Linux/Mac
hello.exe        # Windows

# Output: Hello, World!

‚ö†Ô∏è Common Mistake: Forgetting the semicolon ; at the end of statements! C requires semicolons after every statement. If you get an error, check for missing semicolons first.
