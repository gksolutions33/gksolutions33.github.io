Flex Container

.container { 
  display: flex;
  justify-content: center;
  align-items: center;
}

Flex Direction

.container { flex-direction: row; }     /* horizontal */
.container { flex-direction: column; }   /* vertical */

Flex Properties

.item { flex: 1; }       /* Equal space */
.item { flex-grow: 2; }  /* Grows faster */
.item { gap: 10px; }     /* Space between */
