What is CSS?

CSS (Cascading Style Sheets) is the language for styling web pages. It describes how HTML elements look.

/* CSS is written in rules */
selector {
  property: value;
}

Three Ways to Use CSS

/* Inline */
<p style="color: blue;">Text</p>

/* Internal */
<style> p { color: blue; } </style>

/* External (Best) */
<link rel="stylesheet" href="style.css">

Why CSS?

  • Separates content from design
  • Reuse styles across pages
  • Easy to maintain
  • Create responsive designs
