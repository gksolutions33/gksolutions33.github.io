What is Arduino?

Arduino is an open-source electronics platform used to build projects like IoT systems, robotics, automation, and embedded devices. It is popular because it is easy to learn, affordable, and beginner-friendly.

Arduino Board Types

  • Arduino UNO – Best for beginners
  • Arduino Nano – Small size, breadboard friendly
  • Arduino Mega – Large projects with many pins

Installing Arduino IDE

  2. Download Arduino IDE for your OS

Arduino Program Structure

void setup() { // runs once } void loop() { // runs repeatedly }

LED Blinking Program

void setup() { pinMode(13, OUTPUT); } void loop() { digitalWrite(13, HIGH); delay(1000); digitalWrite(13, LOW); delay(1000); }

Code Explanation

  • pinMode – sets pin direction
  • digitalWrite – turns LED ON/OFF
  • delay – waits in milliseconds

Common Errors

  • COM port not detected → install drivers
  • Upload failed → check board & port
  • LED not glowing → check polarity
